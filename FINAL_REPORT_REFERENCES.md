## Reference Format Notes (Harvard Style Guide)

### Key Rules Applied in This Report:

1. **Author(s)**: Last name, First initial(s).

2. **Date**: Publication year in parentheses.

3. **Title**: In single quotes for articles; italicized for books and reports.

4. **Publication Details**:
   - For journals: *Journal Name*, volume(issue), pp. page range.
   - For books: Publisher name.
   - For conferences: *Proceedings title*, (pp. page range). Organizer/Publisher.

5. **In-text Citations**:
   - Single author: (Smith, 2020)
   - Two authors: (Smith and Jones, 2020)
   - Three or more: (Smith et al., 2020)
   - Multiple sources: (Smith, 2020; Jones, 2019)

6. **Alphabetical Order**: All references listed alphabetically by author surname, then by date (oldest first) if same author.

7. **No Extra Capitals in Titles**: Only capitalize proper nouns and first word after colon.

8. **URLs**: Include if source is primarily online; include retrieval date only if no publication date.
